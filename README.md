# Game-of-Luck

A simple GUI-based game where players generate random numbers between 0 and 10 and compete based on their total score.

## Features
- Random number generation
- Score tracking system
- Interactive GUI

## Technologies Used
- Python
- Tkinter

## How to Run
1. Make sure Python is installed
2. Run the main file:
   python main.py
